**Urheberrechte:**

Der Urheberrechtsschutz für auf GitHub gehosteten Code verbleibt grundsätzlich beim ursprünglichen Autor. Durch das Hochladen von Code auf GitHub gewährt der Autor GitHub eine Lizenz zur Speicherung und Anzeige des Codes. Für Dritte ist die Nutzung, Änderung oder Verbreitung des Codes nur im Rahmen der vom Autor festgelegten Lizenzbedingungen erlaubt. Ohne eine explizite Lizenzierung ist die Nutzung des Codes durch andere nicht gestattet.
